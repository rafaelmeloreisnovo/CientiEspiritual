# Parábolas como Operadores Didáticos

**Estado:** `METHOD_CONCEPTUAL`  
**Fronteira:** parábola não substitui evidência científica, clínica ou histórica.

## 1. Função

Uma parábola permite observar a mesma relação por várias perspectivas sem reduzir a narrativa a uma única moral.

```text
narrativa
→ personagens
→ papéis
→ forças
→ relações
→ invariantes candidatas
→ perguntas testáveis
```

## 2. Conselho das tradições

As tradições são usadas como lentes didáticas, não como caricaturas nem como autoridades científicas automáticas.

| Lente narrativa | Função no ciclo |
|---|---|
| mestre zen | pergunta pela intenção e pelo observador |
| mestre taoísta | observa fluxo, restrição e adaptação |
| escriba hebreu | preserva camadas, origem e interpretação |
| monge cristão | transforma compreensão em serviço e responsabilidade |
| mestre sufi | integra memória, corpo, ritmo e retorno |
| rishi | procura centro, recorrência e multiescala |
| ancião da floresta | lê memória distribuída e interdependência |
| cientista | mede, compara e procura falsificador |
| artista | preserva ambiguidade produtiva sem fingir prova |
| criança | formula a pergunta simples que revela a lacuna |

Nenhuma tradição é reduzida a um símbolo exclusivo; o quadro é uma ferramenta de leitura desta obra.

## 3. Compilação

```text
personagem -> ator
objeto -> estado
movimento -> transição
silêncio -> informação ausente
conflito -> contradição
moral -> hipótese ou princípio metodológico
```

## 4. Exemplo — o sino pequeno

Uma aldeia possuía um sino grande e pequenos sinos no galinheiro. Os pequenos tocaram primeiro, mas ninguém saiu porque seu som parecia insignificante. Quando o sino grande tocou, o risco já havia atravessado o portão.

Invariante didática:

```text
amplitude percebida != relevância operacional
```

Estado inicial:

```text
source_status = DECLARED_BY_AUTHOR

```

Para aplicar a software, aviação, medicina ou sensores, é obrigatório definir amplitude, relevância, população, método e evidência.

## 5. TOKEN_VAZIO

A criança que pergunta “o que ainda não sabemos?” representa uma função epistemológica:

```text
lacuna identificada
→ TOKEN_VAZIO
→ nova observação
```

Não representa ignorância descartável. Representa a recusa de completar a história com uma resposta conveniente.

## 6. Sete leituras

1. seguir a história na ordem;
2. partir do desfecho e voltar às causas;
3. acumular pequenos sinais no tempo;
4. localizar o ponto de mudança;
5. imaginar uma intervenção alternativa;
6. trocar a perspectiva dos personagens;
7. procurar a relação em outras escalas.

## 7. Limites

```text
parábola != diagnóstico
parábola != prova histórica
parábola != mecanismo físico
semelhança narrativa != causalidade
tradição espiritual != dado experimental
```

## 8. Valor operacional

A parábola é útil quando produz:

- uma distinção antes invisível;
- uma contradição registrável;
- um experimento possível;
- uma pergunta discriminante;
- uma ação ética concreta.

Sem uma dessas saídas, ela permanece obra narrativa válida, mas não entra como claim técnico.
